Solid Octo Chainsaw (SOC) mod

mindustry java mod by TheEE145

SOC Mod core and SOC

SOC mod - this mindustry java modular mod with mod depedency and java 16+ code level.

Mod have custom core that have 2 modules: tools and cmdparser (cmdparser like bonus)

Also mod have 2 not core modules: main and annotations

All 4 modules of SOC

Name Wersion Description
tools 1.6 SOC core module
cmdparser 1.0 SOC core command parser
annotations 1.1 SOC mod annotation processors
main 1.4 SOC mod main module
htmlparser (alpha) doesn`t exists will be used as SOC core html parser in future
